\begin{abstract}
Software vulnerability detection using deep learning remains a critical challenge due to two fundamental barriers: the generalisation gap across heterogeneous projects, and the data silo problem that prevents cross-organisation data sharing. In this paper, we propose VulMorph-Fed (Vulnerability Morphology-Guided Federated Learning), a novel federated learning framework designed to enable cross-project vulnerability detection without sharing raw code. The core insight of VulMorph-Fed is that while vulnerability manifestations vary lexically across projects, their structural characteristics (morphology) remain largely invariant. VulMorph-Fed introduces three key components: Vulnerability-Critical Subgraph Abstraction (VCSA) to extract project-invariant structural patterns, Morphology-Conditioned Federated Prototype Aggregation (MCFPA) to share semantic prototypes across clients rather than model parameters, and Morphology-Guided Message Passing (MGMP) to inject global prototype signals into local graph neural network training. We evaluate VulMorph-Fed on a large-scale real-world dataset, demonstrating that it significantly outperforms standard federated learning baselines in cross-project settings, reduces communication costs, and achieves strong privacy-utility trade-offs.
\end{abstract}
